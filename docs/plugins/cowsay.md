Cowsay plugin provides a fun, visual representation of a cow saying the message.

## Usage

```hsf
[hsf3]> load cowsay
 ________________
< Cow here, moo! >
 ----------------
         \   ^__^
          \  (oo)\_______
             (__)\       )\/\
                 ||----w |
                 ||     ||

[i] Use cowsay to call me.
```

## Commands

* `cowsay` - Ask cow to say message.

```hsf
[hsf3]> cowsay 'HatSploit is Great'
 ____________________
< HatSploit is Great >
 --------------------
         \   ^__^
          \  (oo)\_______
             (__)\       )\/\
                 ||----w |
                 ||     ||

```
